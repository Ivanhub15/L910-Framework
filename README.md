# L910-Framework

Минималистичный веб-фреймворк для Node.js, реализованный без использования Express и других фреймворков.

## Команда
- Иван — вариант 3 (Twitch) — тимлид
- Влад — вариант 11
- Маргарита — вариант 12
- Маша — вариант 22

## Возможности фреймворка
- HTTP-сервер на Node.js
- Роутинг (GET, POST, PUT, PATCH, DELETE)
- Middleware (`app.use`)
- Body parser (`req.body`)
- Query params (`req.query`)
- URL params (`req.params`)
- Ответы: `res.send`, `res.json`, `res.status`
- Обработка ошибок

## Пример запуска
```bash
npm install
npm start
